Bizus da Semelhança

AA: Dois ângulos iguais → triângulos semelhantes.

LAL: Dois lados proporcionais e ângulo entre eles igual.

LLL: Três lados proporcionais.

Razões Trigonométricas

sen = cateto oposto / hipotenusa

cos = cateto adjacente / hipotenusa

tan = cateto oposto / cateto adjacente

Ângulos notáveis: 30°, 45°, 60°

Fórmula de Euclides

a = m² - n²

b = 2mn

c = m² + n²

m > n, coprimos, paridades diferentes.

Lei dos Cossenos

a² = b² + c² - 2bc cos(θ)

Para θ = 90° (cos=0), recai no Teorema de Pitágoras.
